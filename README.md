# rop_search.py

A standalone ROP gadget searcher for **rp++** output files.
No external dependencies — pure Python 3 stdlib.

Designed to speed up gadget hunting during the **OSED (EXP-301)** exam and labs,
where you need to quickly find usable primitives across a large gadget dump.

---

## Requirements

- Python 3.6+
- A gadget file generated by **rp++**:

```bash
rp++ -f target.dll -r 5 > gadgets.txt
```

---

## Quick Usage

```bash
# Show best gadget per category (shortest = fewest instructions)
python3 rop_search.py -f gadgets.txt

# Show ALL gadgets per category
python3 rop_search.py -f gadgets.txt -n 0

# Show top-3 gadgets per category
python3 rop_search.py -f gadgets.txt -n 3

# Free regex search
python3 rop_search.py -f gadgets.txt -s "pop eax"

# Filter to specific categories
python3 rop_search.py -f gadgets.txt --pop --xor
python3 rop_search.py -f gadgets.txt --write --deref
python3 rop_search.py -f gadgets.txt --neg

# Statistical summary (how many gadgets per category exist)
python3 rop_search.py -f gadgets.txt --stats

# 64-bit mode (uses r.. register prefix)
python3 rop_search.py -f gadgets.txt -a x86_64

# List all available category flags
python3 rop_search.py -f gadgets.txt --list

# Disable ANSI colors (e.g. when piping to a file)
python3 rop_search.py -f gadgets.txt --no-color
```

## Categories

Each category maps to a set of regex patterns targeting useful ROP primitives.

| Flag        | Category          | OSED Use Case                                      |
|-------------|-------------------|----------------------------------------------------|
| `--write`   | write-what-where  | Arbitrary write primitive (`mov [reg], reg`)       |
| `--deref`   | pointer-deref     | Read from a pointer (`mov reg, [reg]`)             |
| `--swap`    | swap-register     | Copy/move values between registers                 |
| `--pop`     | pop               | Load controlled values from the stack              |
| `--push`    | push              | Push register or memory onto the stack             |
| `--pushad`  | pushad            | Save all registers at once                         |
| `--pushpop` | push-pop chain    | push reg + pop reg in one gadget                   |
| `--inc`     | increment         | `inc reg` or `add reg, 1`                          |
| `--dec`     | decrement         | `dec reg` or `sub reg, 1`                          |
| `--add`     | add               | Add registers or immediate values                  |
| `--sub`     | subtract          | Subtract with or without borrow                    |
| `--neg`     | negate            | `neg` / `not` — useful for encoding shellcode      |
| `--xor`     | xor               | Zero a register or XOR encode/decode               |
| `--and`     | and               | Bitmasking or null-byte removal                    |
| `--or`      | or                | Bitwise OR operations                              |
| `--shift`   | shift             | `shl`, `shr`, `ror`, `rol`, etc.                   |
| `--zero`    | zeroize           | Zero a register (`xor reg, reg` / `sub reg, reg`)  |
| `--pivot`   | eip-to-esp        | Stack pivot for SEH or limited-space exploits      |
| `--nop`     | rop-nop           | ROP NOP sled (`xchg eax, eax; ret`)                |

## OSED Workflow

A typical OSED gadget-hunting session:

```bash
# 1. Generate the gadget file
rp++ -f essfunc.dll -r 5 > gadgets.txt

# 2. Get a quick overview of what's available
python3 rop_search.py -f gadgets.txt --stats

# 3. Find your key primitives
python3 rop_search.py -f gadgets.txt --pop --zero --write --deref

# 4. Need a stack pivot for a SEH chain?
python3 rop_search.py -f gadgets.txt --pivot

# 5. Working around bad characters? Find encoding gadgets
python3 rop_search.py -f gadgets.txt --neg --xor --add

# 6. Verify a specific idea with free search
python3 rop_search.py -f gadgets.txt -s "pop ecx.*pop eax"
```

## Options Reference

| Flag              | Description                                         |
|-------------------|-----------------------------------------------------|
| `-f`, `--file`    | Path to the rp++ gadget file (required)             |
| `-s`, `--search`  | Free regex search across all gadgets                |
| `-n`, `--top`     | Number of gadgets per category (0 = all, default: 0)|
| `-a`, `--arch`    | Architecture: `x86` (default) or `x86_64`          |
| `--stats`         | Print gadget count per category and exit            |
| `--list`          | Print all category flags and exit                   |
| `--no-color`      | Disable ANSI color output                           |

---

## Notes

- Gadgets are ranked by **instruction count** — shorter chains appear first.
- `write-what-where` and `pointer-deref` automatically exclude `[esp]`/`[ebp]`-based gadgets, which are not useful as controlled primitives.
- The `--search` flag accepts full Python regex syntax.
- Colors are auto-disabled when output is redirected to a file.
